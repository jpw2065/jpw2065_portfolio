Artifact

To this date, this has been the project that I have put the most time and effort into. I started out knowing little about game development, and programming. Over my last two years in high school, I was able to develop Artifact from scratch. It was programmed using Java, and the graphics were hand made by me. Cave Game

Cave Game is a simple 2D platformer that I developed. Essentially, it uses a procedurally generated selection of segments that make up the playing field. This project took a month to complete and was programmed using Java. All of the graphics were rendered using a custom rendering method to render to a Java canvas.

Simple 3D

This program is made for users who want a simple 3D modeling program and not yet trained on complicated 3D modeling software. It is the first application I created that used 3D graphics and the mathematics of 3D graphics. To complete this project I studied and applied the mathematics of 3D graphics. It is programmed in Java and rendered using Java's Swing library.

The Pricetool is a project I have been working on at my current job for the past two years. We are using a web application to keep track of media and inventory information. This project involves heavy use of Ruby, JRuby, MySQL, JS, HTML, CSS, and Unix systems.

Students for Action

Students for Action is a website designed to help students get more involved in government and local politics. Although I did not make the website I did help develop their mapping systems. The site includes a map that shows the time and location of posted events. The map uses MySQL along with HTML, and Javascript for a data entry page.
